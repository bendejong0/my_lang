Program     → Main

Main        → MAIN L_PAREN R_PAREN L_CURLY Expression* RETURN RValue SEMICLN R_CURLY

Expression  → Declaration
            | RValue SEMICLN

Declaration → NUM_IDENT IDENT (EQ RValue)? SEMICLN

RValue      → Math
            | NUM_VALUE
            | IDENT
            | List
            | RValue :: RValue

List        → L

Math        → NUM_VALUE (BinaryOperator NUM_VALUE)*
            | UnaryOperator NUM_VALUE

BinaryOperator → PLUS | MINUS | MUL | DIV | MOD

UnaryOperator  → DBL_PLUS | MINUS
